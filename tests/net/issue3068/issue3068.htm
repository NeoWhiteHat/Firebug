Issue 3068: XMLHttpRequests are not shown in Net Panel using XHR filter

  1. Enable and switch to the Net Panel
  2. Reload the page
  3. Clear the Net Panel List
  4. Set "XHR" as filter for the Net Panel
  5. Type "c" (without quotes) into the text field
  6. Set "HTML" as filter for the Net Panel
  7. Enable the checkbox "multipart"
  8. Type "a" (without quotes) into the text field
  • Observed: The request for "c" is shown having the "XHR" filter enabled, the request for "a" appears using the "HTML" filter
  • Expected: Both requests should be shown having the "XHR" filter enabled, because in both cases a XMLHttpRequest is made (see the time is not changing, so the page is not reloaded)
