This test-case is intended for Issue #2160 - Console does not always start auto-scrolled to the bottom.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug, enable & select the Console panel.
  2. Ensure the Console has Show CSS Errors enabled.
  3. Reload the test page.
  4. Scroll the Console panel to the top.
  5. Reload: the content must be again scrolled at the top.
  6. Scroll the Console panel to the bottom.
  7. Reload: the content must be again scrolled at the bottom.
