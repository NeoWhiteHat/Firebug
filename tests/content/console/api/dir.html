Command line API: dir

This test-case is intended for console.dir method.
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Click on the Execute Test button below.
  3. The console panel should display a log (one row) with variable a and value b.


The log in the console panel corresponds to a following expression executed on the Firebug's command line: console.dir({'a': 'b'});
