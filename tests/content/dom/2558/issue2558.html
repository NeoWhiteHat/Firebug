This test-case is intended for Issue #2580 - json object "next" word as object attribute.
Jan Odvarko, odvarko@gmail.com

  1. Open/enable Firebug and refresh.
  2. Press Execute Test button.
  3. Firebug should stop at debugger keyword.
  4. Select the console panel.
  5. Type console.log(arguments) in to the command line and press Enter or Run (in case of multiline command line).
  6. Displayed log should be expandable. If you expand, all values of the arguments array must be displayed, i.e:
    1, "omni", 2
