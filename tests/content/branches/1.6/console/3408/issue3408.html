This test-case is intended for Issue #3408 - Console panel breaks if Firebug is reactivated..
Jan Odvarko, odvarko@gmail.com

  1. Open Firebug and enable the Console panel.
  2. Click on the Execute Test button below. There should be a log in the Console panel.
  3. Deactivate Firebug for the site (click the close button on the FB toolbar).
  4. Activate Firebug again (click the icon in the Fx status bar).
  5. Click on the Execute Test button -> verify the log is there again.
