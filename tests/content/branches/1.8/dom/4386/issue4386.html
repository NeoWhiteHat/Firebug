Issue 4386: DOM breakpoints can't be deleted or disabled/enabled from within the Breakpoints Side Panel

Steps to reproduce

  1. Open Firebug
  2. Enable the Script panel
  3. Reload the page
  4. Switch to the DOM panel
  5. Search for the object screen and expand it
  6. Set breakpoints at availHeight, availWidth and pixelDepth
  7. Switch to the Script panel and there to the Breakpoints side panel
  8. Disable the availHeight and availWidth breakpoint and remove the pixelDepth breakpoint
  9. Switch back to the DOM panel
  10. Scroll down to screen

Observed result

  • The display of the breakpoint for availHeight is updated
  • The display of the breakpoints for availWidth and pixelDepth is not updated

Expected result

  • The display of all DOM breakpoints needs to be updated when they are changed inside the Script panel
