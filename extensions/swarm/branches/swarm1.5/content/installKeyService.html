Installation Instructions for nsIKeyService

Dave Townsend's Mccoy project has a great component that implements nsIKeyService.idl. In Mccoy it is used to sign update.rdf files; we use it to sign swarm web pages.

The keyservice does not come with Firefox; you need to install it.

To install the keyservice you need to download the mccoy XPI file appropriate to your OS, unzip it, then copy the keyservice component files into the FBTest extension component directory. You'll have to restart to load the component. Here are the details:

  1. Download Mccoy according to your OS
  2. Rename the file from .xpi to .zip, then unzip the file
  3. Find the directory "components"
  4. Copy mccoy.dll and mccoy.xpt in to the
  5. Exit and restart the browser
